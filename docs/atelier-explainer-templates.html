Template A

The triangle.

Three corners, three responsibilities. Use when a concept is a new mental model that needs grounding before details.

Applies to · Skills Evaluations

Best when — the topic is a new conceptual model that doesn't map to existing intuitions. The triangle creates the new mental shape.

Template B

The sequence.

Numbered steps top to bottom, code on the right. Use when a concept is a runtime flow — request comes in, things happen in order, response goes out.

Applies to · Memory Runtime
i. Turn arrives, agent opens session. stm.open()
ii. Read short-term context — last twelve turns. stm.get()
iii. If thin, query long-term semantic store. ltm.find()
iv. Compose reply, write turn back. stm.append()

Best when — readers need to follow the flow of a single request through the system. The numbered steps mirror time.

Template C

The network.

Diagram in the middle, panel describing each node and edge. Use when a concept is multiple actors talking to each other.

Applies to · MCP Tool Registry

Best when — the topic involves multiple components with named relationships. Edges are as meaningful as nodes.

Template D

The schema.

Live-feeling table or structure with annotations. Use when the concept is the shape of data — what columns, what keys, what relations.

System rules

Three things that stay constant.

i.

Open with display headline.

Every page opens with a Fraunces italic display headline in the same register as Skills. The template diverges after the opening, not at it. The reader knows where they are before they know what they're reading.

ii.

Cheat sheet where it earns its place.

Templates A and B end with the three-column cheat sheet. Templates C and D do not — for them, the diagram or the schema is the cheat sheet. Don't force a pattern when the page already answers its questions.

iii.

The live strip, always.

Every page closes with a live-state strip tied to the current session. Skills shows what's loaded; Memory shows what's in STM; MCP shows tools fired this turn. The thread that makes the Atelier feel like one surface.
