Above the tabs
The mode strip.
Sits above Architecture / Telemetry / Performance — because patterns aren't a lens, they're the system itself. The tabs underneath show whichever pattern is active.
Two pills
Pattern I, Pattern II.
Roman numerals echo the architecture cards. Each pill shows its name and a small glyph hinting at routing shape — tree for Agents-as-Tools, diamond for Graph.
Right side
Live meta.
Three values that change with the pattern — routing, calls/turn, status. Tells participants what changed when they flip the toggle.

Two patterns, one workbench

What actually changes when you flip the strip.

The chat surface is identical. The inspector tabs are identical. The architecture pages are identical. What flips is how the agent decides which specialist to invoke — and the resulting telemetry shape.
