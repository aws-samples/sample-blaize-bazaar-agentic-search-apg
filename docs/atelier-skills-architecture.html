The mental model

Agents, tools, skills.

Three corners, three responsibilities. Every agent capability is some combination of the three. If you can't tell which one is doing the work, you've muddled them.

WHOAgents.
Autonomous specialists with their own system prompts and decision-making. Blaize ships five: orchestrator, search, recommendation, pricing, support. They reason. They route.
WHATTools.
Discrete, deterministic functions an agent invokes. Typed inputs and outputs, no opinions. semantic_search, get_recommendations, check_inventory, get_pricing, restock.
HOWSkills.
Folders of domain expertise loaded into context when the conversation calls for them. Not a system prompt — small, scoped, conditional. Today: style-advisor and gift-concierge.

The impact

Same agent, same tools, different reply.

The recommendation agent ran identical tool calls in both versions — semantic_search, get_pricing, check_inventory. The skills changed only how the result was handled.

Without skills baseline

Here are three linen pieces in your price range — the Pellier shirt at $148, the Maren tunic at $185, and the Rylan dress at $210.

What's missing No engagement with the gift-ness. No milestone acknowledgment. No fit-tolerance reasoning. No timing or packaging offer. The agent answered the literal query and left every gift-specific decision on the customer.
With style-advisor + gift-concierge +612 tokens

Sixty's a milestone — for that, I'd lean toward something she'll associate with the year, not just the season. Three to consider:

· Maren tunic ($185) — heavy linen, oversized cut, forgiving on size if you're guessing.

· Rylan wrap dress ($210) — natural-fiber blend, ties at the waist so fit is flexible.

· A Pellier shirt + scarf pairing ($176) — if she'd rather have two everyday pieces than one occasion piece.

If size is uncertain, the tunic and the pairing are the safest bets — both fit-tolerant. Want me to flag what's in stock for delivery before the birthday?

What changed gift-concierge named the milestone and surfaced sizing risk. style-advisor shaped each line — heavy linen, ties at the waist, fit-tolerant. Same products, transformed handling.
When in doubt

Three layers, three questions.

AGENTS
Who handles it?
"Which specialist owns this turn?"
  • Has its own system prompt
  • Decides what to do next
  • Can hand off to other agents
  • Add when a new role is needed
TOOLS
What action runs?
"Which function returns the data?"
  • Typed inputs and outputs
  • Deterministic, no opinions
  • Cost: one function call
  • Add when new data or action is needed
SKILLS
How do we think about it?
"Which expert lens loads here?"
  • Markdown + activation contract
  • Loaded conditionally
  • Cost: tokens, only when relevant
  • Add when domain expertise is needed
