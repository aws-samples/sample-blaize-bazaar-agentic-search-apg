What this proves out

Six things that have to land, and how each one does.

Streaming with character.

ai/streamText · CSS keyframes

The shimmer is a moving linear-gradient mask on the text — burgundy sweeps across espresso. Once isStreaming flips false the gradient freezes and the cursor disappears. The serif body keeps it editorial; the burgundy keeps it Blaize.

Thinking, in the agent's voice.

@ai-sdk/react <Reasoning> + Collapsible

The thinking block is shadcn's Collapsible with the AI SDK's reasoning stream wired in. Header reads "Considering · 3.2s", body in italic serif. Collapses by default after the next message — the agent doesn't dwell on its own thinking.

Tool calls, in boutique language.

tool.state === 'result'

The AI SDK exposes tool state per call. We render "Searched the boutique · 12 matched" instead of the raw function name. Pulse indicator while live, green ✓ once complete. Never "calling search_products({"q":"linen"})" — that lives on the Atelier toggle.

Artifacts as products.

Custom message-part renderer

Instead of HTML/JSX in iframes, our artifacts are structured product cards matching the storefront. Stream the JSON, render with the same component the floor uses. Auto-mount on detection with a 380ms scale-from-97% in Framer Motion.

One scroll, not two.

Sticky header + sticky input

Header and input pin; the conversation scrolls between them. Edition strip beneath the header carries the welcome's "Edition 06" mark forward as a one-line continuity cue. No nested scrolls inside thinking blocks — they expand inline.

The Atelier toggle stays separate.

Same agent, two surfaces

Telemetry, raw tool args, latency, embedding dims — all live behind the Atelier switch. Same useChat session, two render trees. Customers see the boutique; engineers see the engine room. The toggle is the only chrome they share.

Built on the same stack you'd reach for. chat-sdk.dev as the starting point — replace defaults with cream, swap Inter for Fraunces in display, repaint the toolcall language. 80% off-the-shelf, 20% editorial discipline.
