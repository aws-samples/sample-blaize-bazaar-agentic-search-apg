The takeaway

Same agent. Same tools. Different answer.

The persona doesn't change what Blaize is — it changes what Blaize knows about you. The variation between the three replies is the workshop's most legible argument for personalization. Three things make the contrast work, and they're the same three things in every personalized AI system.

FROM LTM
History.
Marco's oat tones and August Maren shape the reply. Anna's gift-shaped past shapes the reply. New visitor has none — and the reply is honest about it.
FROM TOOLS
Order context.
When get_order_history returns rows, the agent leans on them. When it returns nothing, the agent reaches for editorial picks instead of fabricating.
FROM SKILLS
Voice.
Style advisor loads on all three. What it produces differs because the agent's underlying context differs. Skills shape the voice; context provides the substance.
